Auto-completion system with AJAX

Upgrade the user experience of your visitors

Please note that script is supported by all recents browsers & Internet Explorer >= 6

Completer.js is a self-completion system developed with HTML (5), CSS 3, JavaScript and PHP. It can be used with or without databases, indifferently systems (MySQL, MariaDB, Oracle, ODBC, ...) Naturally, the script needs a page server to be functional.

How to use ?

Copy/Paste the completer directory on the root of your site. Please note that if you split the files of this directory, your must change configuration datas in php/completer.php and js.completer.js.

First, puts this HTML code in your-file.html.

<div id="searcher" class="form--light-search">  
    <input type="text" name="autocomplete" id="autocomplete" class="input--search" autocomplete="off" />
    <button type="button" name="search" id="search" class="button--search"></button>
</div>

Next, import CSS code from css/completer.css, by a balise link or copy/paste into your own CSS file.

<link href="css/completer.css" rel="stylesheet" type="text/css" media="screen" />

Configure the files/configuration.ini with your login information, the name of the table on which you will perform the query and the columns whose contents will be displayed suggestion. The first column should definitely be the one on which the comparison will be made. A maximum of four columns seems to be reasonable, beyond the display may be unsightly.

If you are not using a database and then query directly a JSON file, only the "enum" is mandatory. It will then necessarily contain the properties of each JSON object you want to display, the first property in the property being informed matcher with expression input by the user.

If you want to change the CSS style of the items displayed, refer to the sheet completer.css style, especially the selectors data-n where n is the order of appearance of the item in the item suggested.

[db]
dsn   = "mysql:host=YOUR_HOST_NAME;dbname=YOUR_DATABASE_NAME;charset=utf8"
user  = "YOUR_USER_NAME"
pwd   = "YOUR_PASSWORD"
table = "YOUR_TABLE_NAME"
enum  = "YOUR_COL_TO_MATCH, YOUR_COL_2, YOUR_COL_3, ..."

Finaly, import the JavaScript file js/completer.min.js into your-file.html, just before the balise </body> :

<script type="text/javascript" src="js/completer.min.js"></script>
